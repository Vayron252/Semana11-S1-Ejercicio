MODIFICADORES

¿Que es?

Es la forma en como se modifica algun elemento o propiedad del elemento seleccionado.

Ejemplos:

textContent

Este tipo de modificador, modifica el texto del elemento.

Se representa así:

const titulo = document.getElementByID('idelemento')

titulo.textContent = 'mi elemento'

innerHTML

Este tipo de modificador, modifica el texto del elemento y puede además añadir código html.

Se representa así:

const titulo1 = document.querySelector('div')

titulo1.innerHTML = '< h1 >Hola< /h >'
