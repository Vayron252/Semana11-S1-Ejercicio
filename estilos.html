ESTILOS

¿Que es?

Es la forma en javascript da stylos css o añade una clase a los elementos html.

Ejemplos:

style

Agrega estilos css al codigo HTML.

Se representa así:

const titulo = document.getElementByID('h1');

titulo.style.background = red;

classList

Agrega clases de la hoja de estilos, al codigo html.

Se representa así:

const titulo1 = document.querySelector('div');

titulo1.classList.add('bg-warning');
