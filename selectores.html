SELECTORES

¿Que es?

Es la forma en como se llama a los elementos dentro del DOM.

Ejemplos:

getElementByID

Este tipo de selector, selecciona al elemento del DOM, siempre y cuando tengan el id al cual se llama.

Se representa así:

const titulo = document.getElementByID('idelemento')

querySelector

Este tipo de selector, selecciona al elemento del DOM y este puede ser por clase, por elemento o por id.

Se representa así:

const titulo1 = document.querySelector('idelemento')

const titulo2 = document.querySelector('#idelemento')

const titulo3 = document.querySelector('h1')
